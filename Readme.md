# IPtoScreen

**IPtoScreen** is a software that **displays** the IP address of the machine on a **Liquid Crystal Display (LCD).**  

## How to set up  

For more information, go to [IPtoScreen Documentation](https://docs.chtrg.fr/liquidcrystal/IPtoScreen).
